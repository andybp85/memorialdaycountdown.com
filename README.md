# How Long Until Memorial Day?

It's the day after Labor Day in New Jersey and this is catharsis.

A static one-page countdown to the next Memorial Day (the last Monday of May), live at
[memorialdaycountdown.com](https://www.memorialdaycountdown.com).

## Structure

- `index.html` — the page; styles inline
- `js/countdown.js` — computes the next Memorial Day and ticks the countdown
- `css/normalize.css` — reset

No build step, no dependencies. Open `index.html` or serve the directory with any static server.
